## Authentication

Every request to the Wachat API must be authenticated using an **API Key**. 
You can generate your key in the **Team Settings** section of your [Dashboard](https://app.wachat.net).

Include the key in your request header as shown below:

```bash
apikey: "YOUR_API_KEY"
